Сборка и запуск проекта "RentalDepartament":
    Проект не требует какой-либо сложной сборки, 
    достаточно лишь иметь установленную visual studio 2019 (и выше) с установленными компонентами для работы с языком программирования (c#).
    Visual Studio сама соберёт проект при попытке запуска через данную среду разработки.
Инструкция для подключения к базе данных:
    Внутри папки есть файл "RentalDepartmentDataBase.backup", который является дампом рабочей базы данных.
    На своём сервере восстановите базу данных по сдампленному файлу.
    Сама база написана на postgreSQL.
    Чтобы подключить проект из среды разработки к базе данных, нужно перейти в класс Program.cs (путь к классу: "RentalDepartment\Program.cs") и внести изменения следующие строковые переменные:
        _dbServerAddress - адрес к базе данных
        _dbPort = - порт для подключения к базе данных
        _dbUserId = - имя пользователя, работающего с базой данных
        _dbPassword - пароль для доступа к базе данных на сервере
        _dbName - название база данных на сервере
    Изменения вносятся согласно данным с вашего сервера, на котором расположена база данных
